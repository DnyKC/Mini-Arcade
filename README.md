# Mini-Arcade

A complete arcade machine built from scratch on custom hardware.
Game logic written in Verilog targeting a Lattice iCE40HX4K FPGA
on a custom PCB. ESP32 handles WiFi leaderboard. 640×480 VGA output.

## Project status
🔨 In progress — currently building intermediate projects

## Hardware
- FPGA: Lattice iCE40HX4K-TQ144 (custom PCB)
- MCU: ESP32-WROOM-32E
- Display: 5" 640×480 IPS VGA panel
- Controls: 8-way joystick + 6 arcade buttons (custom button PCB)
